# Lab Agent — Интерпретация лабораторных анализов

## Что делает агент

Читает письма с PDF-анализами из почтовых папок клиник, интерпретирует каждый показатель через Claude API (claude-opus-4-7) с актуальными референсами 2024–2026, выводит результат в консоль и сохраняет Word-документ в почтовую папку врача.

## Клиники

| Клиника | Почта для чтения | Папка с анализами | Результат |
|---------|-----------------|-------------------|-----------|
| МОЙ МЕД | moimed23@mail.ru | ЛАБОРАТОРИЯ ДИАЛАБ | rezalt LAB moimed |
| ЛайтМед | litemed@mail.ru | ДИАЛАБ ЛАБОРАТОРИЯ_РЕЗУЛЬТАТЫ | litemed LAB rezalt |

Результаты приходят на **marimigi@mail.ru**.

## Требования

- Python 3.10+
- Пакеты: `pip install anthropic pdfplumber python-dotenv imapclient python-docx`
- Файл `.env` с паролями (см. `.env.example`)

### Обязательные переменные .env

```
ANTHROPIC_API_KEY=sk-ant-...
MAILRU_EMAIL=moimed23@mail.ru
MAILRU_PASSWORD=<app-пароль mail.ru>
DEST_EMAIL=marimigi@mail.ru
DEST_PASSWORD=<app-пароль marimigi>
LITEMED_EMAIL=litemed@mail.ru
LITEMED_PASSWORD=<app-пароль litemed>
```

> App-пароли: Настройки mail.ru → Безопасность → Пароли для приложений

## Запуск

### Интерактивный режим (рекомендуется)

Открыть PowerShell в папке проекта и запустить:

```
py lab.py
```

Агент спросит клинику и фамилию пациента.

### С аргументами

```
py lab.py moimed  Иванов
py lab.py litemed Петрова
py lab.py moimed  "Семья Иглина"
```

### Все непрочитанные (пакетный режим)

```
py src/agents/moimedlab/run.py    # МОЙ МЕД
py src/agents/litemedlab/run.py   # ЛайтМед
```

## Поиск пациента

Агент ищет фамилию в **теме письма** (поиск нечувствителен к регистру). Можно передать часть фамилии — агент найдёт все совпадения.

Пример: `Иглин` найдёт письма с темой "Иглина А.В." и "Иглин С.П.".

## Структура вывода

Для каждого найденного письма агент выводит в консоль:

```
============================================================
  Иванов Иван Иванович  |  15.05.2026
============================================================
## Пациент: Иванов И.И. | 45 лет | муж
## Тип анализа: ОАК + биохимия
...
[полная интерпретация]
============================================================
```

И одновременно сохраняет Word-файл (`Интерпретация_Иванов_20260515_1432.docx`) в почтовую папку врача.

```

  agents/

  utils/
    word_builder.py                 — сборка Word-документа

  SKILL.md                          — этот файл
.env                                — секреты (не в git)
.env.example                        — шаблон переменных
requirements.txt                    — зависимости
```

## Использование через Claude Code

Claude Code может запустить агента напрямую:

```bash
# Поиск пациента
py lab.py moimed Иванов

# Пакетный режим
py src/agents/moimedlab/run.py
```

Результат интерпретации появится в консоли и будет виден в ответе Claude.

## Известные ограничения

- Поиск только по теме письма (не по тексту)
- Интерактивный `input()` не работает через терминал Claude Code — используйте аргументы командной строки
- Папки IMAP чувствительны к регистру — значения в `.env` должны совпадать точно
