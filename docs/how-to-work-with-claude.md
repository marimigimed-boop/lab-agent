# Claude-თან მუშაობის სახელმძღვანელო

## ძირითადი პრინციპი
შენ წერ **პრომპტებს** (ინსტრუქციებს). Claude წერს კოდს.
კოდი არ გჭირდება — მხოლოდ ახსენი რა გინდა.

---

## პრომპტების მაგალითები

### ახალი ფიჩერის დამატება
```
დაამატე ფორმა სადაც პაციენტი შეიყვანს სახელს, ტელეფონს და სასურველ თარიღს.
ფორმა უნდა ინახავდეს ბაზაში.
```

### ბაგის გასწორება
```
ჯავშნის გვერდი იხსნება, მაგრამ "შენახვა" ღილაკი არ მუშაობს. გაასწორე.
```

### დიზაინის ცვლილება
```
"ჯავშნის" ღილაკის ფერი შეცვალე მწვანეზე. ღილაკი უნდა იყოს უფრო დიდი.
```

### ახსნა
```
ახსენი რა აკეთებს appointments.py ფაილი
```

## ეფექტური პრომპტების წერა

### კარგი პრომპტი:
```
ჯავშნის ფორმაში დაამატე "სტაციონარი / ამბულატორია" ჩამოსაშლელი სია.
ნაგულისხმევი მნიშვნელობა: "ამბულატორია"
```

### ნაკლებად ეფექტური:
```
გააუმჯობესე ფორმა
```

**რჩევა:** რაც უფრო კონკრეტულია, მით უკეთ. მაგრამ ვაგი პრომპტებიც კარგად მუშაობს — Claude დასვამს დამაზუსტებელ კითხვას.
